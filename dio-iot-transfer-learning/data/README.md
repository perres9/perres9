# Dataset

Coloque as imagens em `data/raw` no formato abaixo:

```text
data/raw/
  classe_1/
    img001.jpg
    img002.jpg
  classe_2/
    img010.jpg
```

Cada pasta representa uma classe.

## Sugestoes de dataset para o desafio

- [Rock Paper Scissors Dataset (TensorFlow)](https://www.tensorflow.org/datasets/catalog/rock_paper_scissors)
- [Intel Image Classification (Kaggle)](https://www.kaggle.com/datasets/puneet6060/intel-image-classification)

Se usar Kaggle, baixe e reorganize as classes dentro de `data/raw`.
